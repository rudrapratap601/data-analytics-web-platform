# 📖 How To Use The Web App

## 1️⃣ Upload Dataset

Go to the **Upload Dataset** page and upload:

- CSV files
- Excel (XLSX) files

The datasets are automatically stored in the PostgreSQL cloud database.

---

## 2️⃣ Explore Dataset

Open the **Data Explorer** page to:

- preview uploaded datasets
- inspect column names
- check datatypes
- view dataset structure

---

## 3️⃣ Clean Dataset

Use the **Dataset Cleaner** page to:

- standardize column names
- convert datatypes
- save cleaned datasets

---

## 4️⃣ Create Relationships

Open the **Relationship Builder** page to connect datasets using matching columns.

Example:

- users.user_id ↔ orders.user_id

This enables:

- multi-table analysis
- JOIN operations
- multi-dataset dashboards

---

## 5️⃣ Perform Analysis

Use the **Dynamic Analysis Builder** to:

- generate SQL-based analysis
- apply aggregations
- perform grouped analysis
- analyze single or multiple datasets

Supported aggregations:

- SUM
- AVG
- COUNT
- MAX
- MIN

---

## 6️⃣ Build Dashboards

Go to the **Dashboard Builder** page to:

- create visual dashboards
- generate interactive charts
- compare datasets
- visualize business insights

Supported charts:

- Bar Chart
- Line Chart
- Pie Chart
- Scatter Plot

---

## 7️⃣ Generate Insights

The platform automatically generates:

- data insights
- trend summaries
- quality reports
- dataset observations

---

# 🌐 Workflow Example

Upload Dataset
→ Create Relationships
→ Run Analysis
→ Generate Dashboard
→ Export Results
